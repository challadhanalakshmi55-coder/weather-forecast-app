PART B: Push to GitHub (3 minutes)
────────────────────────────────────────────────────────────────────────────
1. Create GitHub account (if you don't have one)
2. Create new empty repository called "weather-forecast-app"
3. In Command Prompt, in same folder, run:
   - git init
   - git add .
   - git commit -m "Weather app ready"
   - git branch -M main
   - git remote add origin https://github.com/YOUR-USERNAME/weather-forecast-app.git
   - git push -u origin main
   
   (Replace YOUR-USERNAME with your actual GitHub username)


PART C: Deploy to Streamlit Cloud (5 minutes)
────────────────────────────────────────────────────────────────────────────
1. Go to: https://streamlit.io/cloud
2. Sign up with GitHub → Authorize
3. Click "New app"
4. Fill in:
   - Repository owner: YOUR-USERNAME
   - Repository: weather-forecast-app
   - Branch: main
   - Main file path: app.py
5. Click "Deploy" → Wait 2-5 minutes


PART D: Add API Key (2 minutes) ⚠️  IMPORTANT!
────────────────────────────────────────────────────────────────────────────
1. After deployment, click ☰ menu (top right of app)
2. Click "Settings" → "Secrets"
3. Paste: OPENWEATHERMAP_API_KEY = "9aac7ae37f68e3d23dbfcc5f0fa4a1ff"
4. Click "Save"
5. Wait 30 seconds for reboot


PART E: Test Live App (1 minute)
